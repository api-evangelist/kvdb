# KVdb (kvdb)
Stop wasting time setting up NoSQL databases. KVdb is a hosted serverless key-value database with a simple HTTPS REST API. Buckets act as namespaces for keys, with built-in access control via secret, read, and write keys, custom Lua scripts, and per-bucket TTLs.

## APIs

### KVdb
KVdb provides a hosted, serverless key-value database accessible over a simple REST API. Operations include creating and managing buckets, setting and retrieving values, atomic numeric increments, prefix-based key listing, and uploading and executing custom Lua scripts.
